Arrays

Arrays are used to store multiple values under one variable

Creating an array

The following code would create a script

    const nameOfArray = [];
    nameOfArray[0] = value1;
    nameOfArray[1] = value2;
    nameOfArray[2] = value3;

    console.log(nameOfArray);

        > [value1, value2, value3]

The following works just as well

    const nameOfArray = [value1, value2, value3];

Looping an array

An array can be looped through with a regular for-loop or a for-of-loop

for-loop

    const nameOfArray = [value1, value2, value3];

    for (let i = 0; i < nameOfArray.length; i++) {
        console.log(`Value: ${nameOfArray[i]}`);
    }

for-of-loop

    for (let item of nameOfArray) {
        console.log(`Value: ${item}`);

Array methods

Arrays can be modified with an assortment of predefined methods

    nameOfArray.sort();

Sorts the array alphabetically

    nameOfArray.sort((a,b) => a-b);

Sorts the array numerically

    nameOfArray.reverse();

Reverses the order of the items in the array 

    nameOfArray.shift();

Deletes and returns the value of the first item in an array

    nameOfArray.pop();

Deletes and returns the value of the last item in an array

    nameOfArray.push(value);

Adds the value as a new item to the end of the array, more convenient
than the push technique at the top of the page

    nameOfArray.includes(value);

Returns True if the provided value is found in the array

Object literals (a.k.a. dictionaries)

Object literals can be used to store multiple values under one variable, but also to
link those values to a name. This allows for a sort of variable within a variable

    const arrayName = {
        name: 'Steve',
        address: '214 Royal Lane',
        phoneNumber: '040 6745286',
    };
